# Crucible

**Crucible** is a Paper plugin that adds a fully custom drug and chemical processing system, with realistic production chains, quality scaling, tolerance, crashes, and more.

---

## Features

### Chemical Processing
- Custom machines (e.g. **Chemical Station**, **Drying Rack**, **Grinding Press**)
- Time-based processing
- Multi-step production chains

### Drugs & Effects
- Each drug has:
    - **Quality tiers (★ to ★★★★★)**
    - **Tolerance build-up**
    - **Varying effects**
    - **Crashes / withdrawals**
- Some drugs have:
    - **Unique buffs/debuffs**
- Effects scale with quality (duration, intensity, penalties)

### Crafting & Processing

- All recipes are custom-controlled
- Vanilla recipes are blocked where necessary
- Furnace / smoker behaviour is overridden safely
- Custom items are identified via PersistentDataContainer, not materials

---

## Compatibility

- **Minecraft** 1.21.11 (Paper / Spigot)
- **Geyser + Floodgate**
